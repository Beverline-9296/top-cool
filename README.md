## Getting Started

1. **Local Development**: Simply open `index.html` in any modern web browser
2. **Web Hosting**: Upload all files to your web hosting provider
3. **Customization**: Edit the content in `index.html` to match your business details

## Customization Guide

### Business Information
Edit these sections in `index.html`:

1. **Company Name**: Change "AlumGlass Pro" throughout the file
2. **Contact Information**: Update phone, email, and address
3. **Business Hours**: Modify the hours in the contact section
4. **Services**: Customize the services offered
5. **About Section**: Update the company description

### Styling
- Colors can be changed in `styles.css` by modifying the CSS custom properties
- The primary blue color (#2563eb) can be replaced with your brand colors
- Fonts can be changed by updating the Google Fonts import

### Content
- Replace placeholder text with your actual business information
- Add real project images to the portfolio section
- Update service descriptions to match your offerings
- Customize the hero section messaging
